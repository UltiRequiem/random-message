# ranmess

Get random quotes in terminal.

This project makes Fetch of the [Quotable.io API](https://api.quotable.io/random).

![Screenshot](./assets/screenshot.png)

## Install

`npm -g i ranmess`

## Usage

```bash
ranmess
```
